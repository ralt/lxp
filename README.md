# lxp

## Usage

Relies on an [lxpfile](t/lxpfile) to define a list of
containers to use in a project. Based on lxc.

## Installation

Development mode only.

```
$ git clone https://github.com/ralt/lxp ~/common-lisp/lxp
$ cd ~/common-lisp/lxp
$ make
# Go to a folder where lxpfile is...
$ ./lxp
```
